# REQUIRED ASSETS

## 1. Store Icon (128x128 PNG)
- Must be exactly 128x128 pixels
- PNG format
- Use the SVG in /extension/icons/icon.svg as source
- Convert using: https://cloudconvert.com/svg-to-png

## 2. Screenshots (1280x800 or 640x400)
You need at least 1, recommended 5 screenshots showing:

Screenshot 1: "Main popup showing clipboard history"
- Open the extension popup
- Have 3-4 sample clips visible
- Show the search bar

Screenshot 2: "Workspaces for organization"
- Show different workspace tabs
- Demonstrate organizing clips

Screenshot 3: "Templates feature"
- Show the templates modal
- Have 2-3 sample templates

Screenshot 4: "Image clipboard support"
- Show an image in the clipboard history
- Demonstrate image capture

Screenshot 5: "Settings and customization"
- Show the settings page
- Highlight privacy features

### How to take screenshots:
1. Open Chrome DevTools (F12)
2. Click device toolbar (Ctrl+Shift+M)
3. Set dimensions to 1280x800
4. Navigate to extension popup
5. Screenshot (Ctrl+Shift+P ‚Üí "Capture screenshot")

# PUBLISHING STEPS

1. Go to: https://chrome.google.com/webstore/devconsole
2. Pay $5 one-time developer fee
3. Click "New Item"
4. Upload ZIP file (clipstash-extension.zip)
5. Fill in store listing details (copy from above)
6. Upload icon and screenshots
7. Select category: Productivity
8. Add privacy policy URL
9. Submit for review

Review typically takes 1-3 business days.
